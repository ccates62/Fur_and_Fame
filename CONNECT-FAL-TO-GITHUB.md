## Step 1: Disconnect Old Account from GitHub

1. Go to: https://github.com/settings/applications
2. Click **"Authorized OAuth Apps"** tab
3. Find **"fal.ai"** in the list
4. Click on it to view details
5. Click **"Revoke access"** button
6. Confirm the revocation

This disconnects your OLD fal.ai account from GitHub.

---

## Step 2: Connect New fal.ai Account to GitHub

1. Go to: https://fal.ai/dashboard
2. Make sure you're logged into your NEW account (ccates.timberlinecollective@gmail.com)
3. Look for a **"Connect GitHub"** or **"Sign in with GitHub"** option
   - This might be in Settings â†’ Integrations
   - Or in Account Settings
   - Or when you first log in, it may prompt you
4. Click the GitHub connection option
5. Authorize fal.ai to access your GitHub account
6. Select the permissions you want to grant

---

## Step 3: Verify Connection

1. Go back to: https://fal.ai/dashboard
2. Check that your GitHub account is connected
3. You should see your GitHub username/avatar in the fal.ai dashboard

---

## Alternative: Sign In with GitHub

If fal.ai supports "Sign in with GitHub":
1. Log out of your current fal.ai account
2. Go to: https://fal.ai/dashboard
3. Click **"Sign in with GitHub"**
4. Authorize the connection
5. This will create/link your account

---

**Note:** If you can't find the GitHub connection option in fal.ai, it might not be available or might be in a different location. Check their documentation or contact support.
